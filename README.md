# [RUS]
# Радиометр на базе Arduino с мобильным приложением Android

Видео от разработчика исходного устройства: https://www.youtube.com/watch?v=lN3ZoD0i_3s

# Изменения оригинальной прошивки

  - Добавлена поддержка Bluetooth
  - Изменен вид интерфейса
  - Оптимизирован поисковый режим
  - Поддержка двух счётчиков Гейгера-Мюллера
  - Установлено разрешение экрана 128x64

# В работе

  - Управление звуковым оповещением прибора через приложение

# Возможности мобильного приложения

 - Отображение текущего уровня излучения
 - Отображение усреднённого уровня за последнее время
 - Режим измерения
 - Управление пороговыми значениями тревоги
 - Отправка команд управления звуком устройства
